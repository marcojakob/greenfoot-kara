------------------------------------------------------------------------
-- Kara Sokoban (ENGLISH) --
------------------------------------------------------------------------

A classic Sokoban game with Kara.
The goal of the game is to push all the mushrooms on the leafs.

PROGRAM SOURCE REMARKS:
-   All the programming for Kara (and the Highscore) is to be made in the 
    class MyKara.
-   There is a developer-mode to skip the main menu (can be set in MyKara).
-   The Highscore can be enabled/disabled in MyKara.
-   The icons for the actors and the background can be changed by 
    right-clicking an Actor or the GameScreen --> setImage...
-   To create new levels, one can create new instances of the actors with 
    the mouse and place them in the world (developer-mode should be used). 
    When finished, right-clicking on the world --> generateASCIILevel() 
    will print the level to the console. The generated text can then be
    copied into the levels text file.

-- Kara Sokoban (GERMAN) --
------------------------------------------------------------------------	

Dies ist ein klassisches Kistenschieb-Spiel mit Kara.
Das Ziel des Spiels ist, alle Pilze auf die Blätter zu schieben.

BEMERKUNGEN ZUM QUELLTEXT:
-	Die gesamte Programmierung von Kara (und der Highscore) sollte in der
	Klasse MyKara gemacht werden.
-	Es gibt ein Entwickler-Modus (developer-mode), um das Hauptmenu zu
	überspringen. Dies kann in der Klasse MyKara eingestellt werden.
-	Die Highscore kann in MyKara aktiviert/deaktiviert werden.
-	Die Bilder für alle Actors oder den GameScreen können geändert werden
	mit Rechtsklick --> setImage...
-	Um neue Levels zu erstellen kann man zuerst die Actors mit der Maus auf
	der Welt platzieren (im Entwickler-Modus). 
	Wenn man den Level fertig designed hat, kann man mit Rechtsklick auf die
	Welt --> generateASCIILevel() den Level auf die Konsole ausgeben lassen.
	Der generierte ASCII Text kann anschliessend in die levels Textdatei
	kopiert werden.
